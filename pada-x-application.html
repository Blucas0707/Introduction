個人簡介

目前任職於半導體業，是一名製程工程師。
在學生時期就接觸過程式，最初是從Ｃ語言入門，到研究生涯的Matlab，再到自學的Python & SQL。因此對程式產生了不小的興趣，而這份火花一直在心底，不曾熄滅。
在工作上使用VBA ,SQL & Python來撈取data，和後續分析，而在工作之餘也寫過爬蟲，來學習如何爬取資料，也幫朋友架過Line bot，來作為社團間資訊分享的方式。
目前發現各產業都對數位轉型與網路行銷的人才有著滿大的缺口，而這也暗和我在網站工程師的不斷學習的目標，因此決定參加此計劃，來精進自身能力，尋求下一個職業發展。

成為網站工程師的動機

起初只覺得爬蟲很酷，但隨著深入了解，才發現需要大量前後端的網站架設知識。
同時，因為家中從事傳統產業，近年來也在尋求數位轉型，對於網站的建置、數位行銷甚至到通路金流串接等，都需要相關網站知識。
曾經試著用Wordpress + Siteground 來製作網站，但發現模板方便的同時，也帶來極大的侷限，而對於前後端串接的一知半解，更是可能會有潛在的風險與維護困難。
而我自己曾經試過串接line Message API 及使用Heroku PaaS的經驗，同時也因為工作所需，對於SQL也有初步的認識，所以原先是想走後端工程師。
然而，現在的我，更想將前端的知識ㄧ併整合，未來目標成為一位全端工程師！
